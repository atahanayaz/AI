Part 4: Fuzzy Logic

Restaurant Tip Calculator - Dataset #22

Problem: How much tip to give?

Inputs:

  • Food Quality (0-10)
  • Service Quality (0-10)

Output:

  • Tip Percentage (0-30%)

Why Fuzzy Logic?

Human decisions are imprecise: "Food was pretty good" ≠ exact rating

Smooth transitions, no sharp boundaries ✓

Fuzzy Logic System Design

Fuzzy Sets:

  • Food: Poor, Average, Excellent
  • Service: Poor, Average, Excellent
  • Tip: Low (0-15%), Medium (10-20%), High (15-30%)

Fuzzy Rules: 3×3 = 9 rules

Example:

IF Food is Excellent AND Service is Excellent

THEN Tip is High

Fuzzy Membership Functions

Triangular functions with smooth overlapping regions

Fuzzy Logic Output Surface (3D)

Shows tip % for all input combinations - smooth transitions ✓

Fuzzy Logic - Contour Map

Test Results: Poor/Poor → ~7% | Excellent/Excellent → ~25%
