Knapsack Optimization (Dataset #7)

Problem Specification

  • Items: 10 items with different weights and values
  • Capacity: 53 kg maximum
  • Objective: Maximize total value while respecting capacity

Implementation Details

  • From scratch: No GA libraries used
  • Population: 10, Generations: 100
  • Operators: Roulette selection, single-point crossover, bit-flip mutation
  • Elitism: Best solution always preserved
